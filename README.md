LumoRead (ระบบสนับสนุนการอ่านสำหรับนักเรียนประถมต้น)
โปรเจกต์นี้คือ LumoRead เป็นเว็บแอปพลิเคชันที่พัฒนาขึ้นมาเพื่อช่วยนักเรียนระดับประถมต้น (ป.1 - ป.3) ฝึกอ่านออกเสียงภาษาไทย โดยมีระบบ AI (Speech-to-Text) เข้ามาช่วยประเมินความถูกต้องของการออกเสียงและให้คำแนะนำแก่นักเรียนได้ทันที

รายละเอียดของระบบ
ระบบถูกออกแบบมาให้ใช้งานได้ 3 กลุ่มหลัก:

นักเรียน: สามารถฝึกอ่านและทำแบบทดสอบได้หลายรูปแบบ มีระบบเสียงต้นแบบ (Text-to-Speech) ให้กดฟังก่อนเริ่มฝึก และสามารถดูประวัติคะแนนของตัวเองได้

คุณครู: มีระบบสำหรับจัดการแบบฝึกหัดและคลังคำศัพท์ สามารถติดตามพัฒนาการการอ่านของนักเรียนได้ทั้งรายบุคคลและรายกลุ่ม พร้อมทั้งให้คำแนะนำเพิ่มเติมแก่นักเรียน

แอดมิน: ทำหน้าที่ดูแลจัดการบัญชีผู้ใช้งานและอนุมัติการสมัครสมาชิกในระบบ

เทคโนโลยีที่ใช้
Frontend: พัฒนาด้วย Flutter Web ออกแบบให้เหมาะกับการใช้งานของเด็ก (ตัวหนังสือใหญ่ ปุ่มกดง่าย)

Backend: ใช้ Flask สำหรับจัดการข้อมูลเสียงและเชื่อมต่อกับโมเดล AI

Database: ใช้ Firebase Firestore ในการเก็บข้อมูลผู้ใช้และผลคะแนนต่างๆ

AI Model: ใช้ Wav2Vec2 (GUSGUSxCAT/EDUPJ2) ที่ผ่านการ Fine-tuning เพื่อให้รองรับเสียงภาษาไทยและเสียงเด็กโดยเฉพาะ
